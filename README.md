# Solid Webflow

Custom code of Solid Webflow website.

## Getting Started

Run development server:
```bash
pnpm dev
```

## Contribute

Validate the code before committing:
1. Lint:
  ```bash
  pnpm lint
  ```
2. Format:
  ```bash
  pnpm format
  ```
3. Check:
  ```bash
  pnpm check
  ```

## Deploy

Build the project:
```bash
pnpm build
```
